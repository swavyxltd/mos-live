# PWA Icons

This directory should contain the following icon files for the Progressive Web App:

- `icon-72x72.png` - 72x72 pixels
- `icon-96x96.png` - 96x96 pixels
- `icon-128x128.png` - 128x128 pixels
- `icon-144x144.png` - 144x144 pixels
- `icon-152x152.png` - 152x152 pixels
- `icon-192x192.png` - 192x192 pixels
- `icon-384x384.png` - 384x384 pixels
- `icon-512x512.png` - 512x512 pixels

All icons should be PNG format and ideally include both "any" and "maskable" versions for better compatibility across devices.

The manifest.json file references these icons. Once you add the icon files, the PWA will be fully functional.
